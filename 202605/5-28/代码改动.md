# 代码改动

## 基线约束

本次代码改动只允许服务于：

```text
#/ -> Stented
```

不得破坏：

- `#/`
- `#/category/stented`
- `#/valve/hancock-ii`
- `#/valve/hancock-ii/size/21`
- `#/valve/hancock-ii/size/21/thv/current`
- `#/valve/hancock-ii/size/21/thv/current/plan/evolut`
- `#/valve/hancock-ii/size/21/thv/current/plan/evolut/placement`

不得修改：

- 医学内容
- `scripts/data/valves.js`
- 深链页面业务逻辑
- 路由系统

## 4. `tests/navigation.test.mjs`

### 4.1 Home 不允许 fallback

代码：

```js
assert(!homeHtml.includes("未找到页面"), "Home should not render the route fallback");
```

基线目的：

```text
防止 #/ 被 fallback 页面替代。
```

### 4.2 Home 主链入口区必须存在

代码：

```js
assert(homeHtml.includes("data-home-main-flow"), "Home should expose the primary main-flow group");
```

基线目的：

```text
防止 Home 主分类区被误删或重构丢失标记。
```

### 4.3 Stented 链接必须存在

代码：

```js
assert(homeHtml.includes('href="#/category/stented"'), "Home should expose the Stented main-flow entry");
```

基线目的：

```text
防止 #/ -> Stented 入口断链。
```

### 4.4 Stented 必须是主链入口

代码：

```js
assert(homeHtml.includes('data-main-flow-entry="stented"'), "Home should mark the Stented row as the main-flow entry");
```

基线目的：

```text
防止 Stented 入口失去主链标记。
```

### 4.5 主链入口必须唯一

代码：

```js
assert(
  (homeHtml.match(/data-main-flow-entry=/g) || []).length === 1,
  "Home should keep Stented as the only explicit main-flow entry",
);
```

基线目的：

```text
防止多个入口被错误标成主链入口。
```

### 4.6 Home 壳层必须保留

代码：

```js
assert(homeHtml.includes('class="bottom-tabs"'), "Home should keep the native bottom nav");
assert(homeHtml.includes('class="bottom-tab is-active" href="#/"'), "Home should keep the Home tab active");
assert(homeHtml.includes('class="floating-quick"'), "Home should keep the quick selector trigger");
assert(homeHtml.includes("screen--native-shell"), "Home should render inside the native screenshot shell");
assert(homeHtml.includes("page--single-column"), "Home should stay single-column in screenshot mode");
assert(homeHtml.includes("topbar__native-title"), "Home should keep the compact native title");
```

基线目的：

```text
防止 Home 壳层、bottom tabs、Quick Selector 被破坏。
```
